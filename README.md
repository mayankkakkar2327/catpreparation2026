# CATPreparation2026

SEO-first static MVP for `catpreparation2026.com`, an independent CAT 2026 preparation, coaching, IIM, MBA college, and MBA entrance exam information platform.

## What is included

- Static generated HTML pages
- CAT 2026 information pages
- CAT coaching directory with filters
- City-wise coaching pages
- Coaching profile pages
- IIM, MBA college, preparation, and trust pages
- JSON-LD structured data
- Sitemap and robots.txt

## Local commands

```bash
npm run build
npm run start
```

The preview server runs at `http://localhost:4173` by default.

## Vercel

The project is configured as a static deployment. Vercel should run:

```bash
npm run build
```

and serve the generated `site` directory.
